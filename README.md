# Medicenter_Flexbox
Esse projeto de template de site foi desenvolvido durante as aulas do curso da B7Web. Foram utilizados conceitos básicos de HTML e CSS, com enfoque ao uso do CSS Flexbox.

Também foram utilizados conceitos de responsivdade através do recurso CSS @media e efeitos báiscos em JavaScript para que o funcionasse de forma responsiva.
